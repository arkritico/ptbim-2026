# 6.º ptBIM 2026 — App

App de bolso para o **6.º Congresso Português de BIM** (FEUP · 17–19 jun 2026), com identidade **Wallnut** a preto-e-branco, alinhada com a imagem do evento.

**▶ Abrir:** https://arkritico.github.io/ptbim-2026/

<p><img src="qr.png" alt="QR" width="180"></p>

## O que faz
- **Programa completo** — Agenda (por dia), Grelha (por sala) e Plano (estratégia, top-5, conflitos, networking).
- **Marcar sessões** assistidas e **escrever notas** por sessão.
- **Fotos por sessão** — câmara ou galeria — com **correção de perspetiva** (arrasta 4 cantos → endireita o slide). Ideal para fotografar projeções.
- **Local** — mapa do campus FEUP, como chegar (metro/STCP), salas, contactos e links (programa PDF, ptbim.org).
- **Privado** — notas e fotos ficam **só no telemóvel** (localStorage + IndexedDB). **Export/Import** JSON para backup ou levar para outro aparelho.
- **Instalável (PWA)** — “Adicionar ao ecrã principal” (Android/iOS). Funciona **offline** após o 1.º acesso.
- **Partilha** — QR embutido no separador *Local*.
